##  Contenido del Proyecto

El taller se divide en tres secciones clave:

### 1.  Operaciones con NumPy
- Manejo de arreglos multidimensionales.
- Generación de matrices con datos aleatorios.
- Aplicación de métodos estadísticos (Media, Mediana, Desviación Estándar).

### 2.  Manipulación de Datos con Pandas (Daframes)
- Carga y exploración de datos estructurados.
- Diferencias clave entre `loc` (etiquetas) e `iloc` (posiciones).
- Filtrado avanzado con condiciones múltiples, `.isin()` y `.str.contains()`.
- Generación de métricas de interés (Top 10 contenido popular, series longevas, etc.).

### 3.  Procesamiento de Texto (Drácula - Project Gutenberg)
- Descarga automatizada de archivos con `urllib`.
- Limpieza de texto mediante **Expresiones Regulares (Regex)**.
- Tokenización y conteo de frecuencias con `collections.Counter`.
- Medición de tiempos de ejecución y exportación de reportes (.txt).

## 🛠️ Herramientas Utilizadas
- **Python 3.x**
- **Pandas**: Para la gestión de DataFrames.
- **NumPy**: Para cálculos numéricos eficientes.
- **Re**: Para limpieza de texto.
- **Urllib**: Para peticiones web.
